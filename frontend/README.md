# AI-JARVIS Frontend

Next.js dashboard for AI-JARVIS

## Features

- Real-time voice visualization
- Live webcam feed with vision overlay
- Action execution logs
- Memory browser
- Settings panel
- Dark mode

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Environment Variables

See `.env.example` in root directory.

## Tech Stack

- Next.js 14
- React 18
- TailwindCSS
- WebSockets
- Chart.js
